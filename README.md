# 🏆 NIKE TOMA EL JUEGO 2026 🏆

¡Bienvenido al repositorio oficial del campeonato "NIKE TOMA EL JUEGO 2026"! Este proyecto contiene las páginas de inscripción e información para las diferentes sedes del torneo.

## 🚀 Descripción

Este proyecto web está diseñado para gestionar y presentar la información de un torneo de fútbol que se lleva a cabo en diversas localidades de Lima, Perú. Cada sede tiene su propia página de presentación, y además, cuenta con una sección de administración para gestionar los datos del evento.

## ✨ Características

-   **Páginas por Sede:** Páginas dedicadas para cada una de las sedes del torneo:
    -   Callao
    -   Chorrillos
    -   La Victoria
    -   San Miguel
    -   Surquillo
-   **Sección de Administración:** Un panel de administrador para la gestión del campeonato.
-   **Recursos Descargables:** Acceso a documentos importantes como autorizaciones y cesiones de derechos.
-   **Diseño Atractivo:** Interfaz de usuario con el branding de Nike Futbol.

## 📁 Estructura del Proyecto

El repositorio está organizado de la siguiente manera:

```
├── Admin/              # Panel de administración
├── Archivos/           # Documentos y autorizaciones
├── Callao/             # Página de la sede Callao
├── Chorrillos/         # Página de la sede Chorrillos
├── CSS/                # Hojas de estilo
├── fonts/              # Tipografías utilizadas
├── IMAGES/             # Gráficos y logos del torneo
├── JS/                 # Scripts para la interactividad
├── LaVictoria/         # Página de la sede La Victoria
├── SanMiguel/          # Página de la sede San Miguel
└── Surquillo/          # Página de la sede Surquillo
```

## 🛠️ Tecnologías Utilizadas

-   **HTML5**
-   **CSS3**
-   **JavaScript**

## 🏁 Cómo Empezar

Para visualizar y utilizar este proyecto, simplemente clona este repositorio en tu máquina local y abre los archivos `index.html` de cada sede en tu navegador web.

```bash
git clone <URL-DEL-REPOSITORIO>
```
